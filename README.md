# ReformaAI

ReformaAI es una aplicación web que permite visualizar reformas usando inteligencia artificial.

## Qué hace este proyecto

- Sube imágenes de espacios
- Aplica modificaciones con IA
- Genera visualizaciones de reformas

## Requisitos

- Node.js instalado
- Cuenta en Replicate.ai
- API Token de Replicate

## Instalación

En la carpeta del proyecto:

npm install

## Configuración

Crear un archivo .env con:

REPLICATE_API_TOKEN=TU_TOKEN_AQUI

## Ejecutar el proyecto

node server-reforma-ai.js

## Notas importantes

- No subir nunca el archivo .env a Github
- No subir la carpeta node_modules
